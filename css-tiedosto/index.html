HARJOITUS 2. CSS-tiedostoon tutustuminen


HTML ja CSS validointi

Validointi tarkoittaa, että tarkistamme HTML- ja CSS-tiedostot virheiden varalta.

Validaattori tarkistaa annetut tiedostot, ja ilmoittaa mahdollisista virheistä joita ohjelmoija ei ole itse huomannut.

Linkkejä validaattoreihin

https://validator.w3.org/

https://jigsaw.w3.org/css-validator/

https://www.freeformatter.com/html-validator.html
